# IFC4x3 Alignment Geometry Implementation Guide

This repository contains an implementation guide for developers adding IFC4x3 alignment geometry support to infrastructure software applications. It captures lessons learned while implementing alignment geometry in [IfcOpenShell](https://www.ifcopenshell.org) and provides mathematical reference for all horizontal, vertical, and cant curve types defined in IFC4x3.

See the [Table of Contents](TableOfContents.md) to get started.
